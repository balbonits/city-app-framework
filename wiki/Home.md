# City App Framework Wiki

Welcome to the **City App Framework** - the world's first AI-native development framework that enables solo developers to build applications with enterprise-level capabilities through AI collaboration.

## 🏙️ Framework Overview

The City App Framework uses a city governance metaphor where the developer is the "mayor" and AI agents are specialized "citizens" who build, maintain, and evolve applications.

### Key Features
- **AI-Native Architecture**: Every component designed for AI agents first
- **Claude Code Optimization**: Rich context files from day one
- **Quality Defaults**: TypeScript strict mode, testing, accessibility built-in
- **Professional Templates**: Blog/CMS, Dashboard, E-commerce, Minimal
- **Anti-Overengineering**: Explicit guidelines to prevent complexity creep

## 🚀 Quick Start

```bash
# Install CLI globally (when published)
npm install -g create-city-app

# Or use locally for development
git clone https://github.com/your-username/create-city-app
cd create-city-app
npm install && npm link

# Generate a project
npx create-city-app my-blog --template=blog --yes
cd my-blog
npm install
npm run dev

# Start AI-assisted development
npm run ai:start
```
